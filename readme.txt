Adds missing alt text to WooCommerce product images.

== Description ==

WooCommerce Alt Text is a plugin that adds missing alt text to WooCommerce product images. The plugin works by analyzing the content of the page and comparing it to a list of predefined keywords.

== Installation ==

Upload the woocommerce-alt-text folder to the /wp-content/plugins/ directory.
Activate the plugin through the 'Plugins' menu in WordPress.
Go to WooCommerce Alt Text Settings to configure the plugin.
== Usage ==

After activating the plugin, it will automatically add missing alt text to your WooCommerce product images.

== Settings ==

The WooCommerce Alt Text Settings page allows you to configure the plugin.

Alt Text Source: Choose the source for the alt text (filename or H1 title).
Apply Alt Text Update: Choose whether to apply the alt text update to all images or only to missing alt text.
